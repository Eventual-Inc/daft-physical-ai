# Examples

A ready-to-run hand-tracking demo, generated by the `daft-physical-ai` CLI.

- **`demo.py`** / **`demo.ipynb`** - read a LeRobot dataset, run `track_hands`
  (MediaPipe, CPU), and show the keypoints. Same steps, script and notebook.

Run it:

```bash
pip install "daft-physical-ai[mediapipe]"
python examples/demo.py
# or: jupyter lab examples/demo.ipynb
```

Want a different setup (WiLoR, both methods, or a Modal GPU runtime)? Generate
your own with the CLI:

```bash
daft-physical-ai            # interactive
daft-physical-ai --method wilor --runtime modal --mano-path MANO_RIGHT.pkl --no-input
```
