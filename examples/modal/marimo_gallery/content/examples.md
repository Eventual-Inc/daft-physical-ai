# Examples

Examples should make the repository value visible before a researcher reads API
reference. Each card should show a concrete physical-AI workflow: the input
data, the operation, and the result.

Start with a small set of examples and make each one deep enough to be useful.

## Initial example set

- DROID episode indexing with Daft and Marimo.
- EgoDex hand tracking with rendered keypoints and a local run path.
- Private-data handoff for users who need to run on their own datasets.

Future examples can add video clips, query plans, dataset schemas, and
evaluation artifacts without changing the website architecture.
